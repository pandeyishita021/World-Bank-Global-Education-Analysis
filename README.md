This project performs Exploratory Data Analysis (EDA) on global education data using the World Bank EdStats dataset. The dataset was cleaned, reshaped from wide to long format, and merged with country metadata (region and income group) to enable meaningful comparisons.
The analysis focuses on two key indicators: Government Expenditure on Education (% of GDP) and Pupil–Teacher Ratio to understand global education investment and classroom conditions.
Using Python, Pandas, Matplotlib, and Seaborn, the project explores global trends, regional differences, and income-based disparities in education systems.
Key Insight: Countries with higher education spending often show better classroom conditions, while low-income countries tend to have higher pupil–teacher ratios.
